# 程式是什麼？

## Computer Program

是指一組指示電腦或其他具有訊息處理能力的電子裝置<mark>每一步動作的指令</mark>，通常用某種程式設計語言編寫，執行於某種目標體系結構上。

* 在電腦上展示一行Hello World
* 在Android上執行傳說對決
