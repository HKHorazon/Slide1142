# 設定 App Icon

做好的 App 不想要預設的安卓機器人圖示？

1. 準備一張方形的圖片 (png 或 jpg)，最好 512x512 像素，不要太大。
2. 上傳到 Media。
3. 點選 **Screen1** 元件。
4. 在右邊屬性找到 **Icon**，選剛剛那張圖。

這樣安裝到手機上就會顯示你的圖示了！

# 打包 APK (Build)

之前的 AI Companion 只是測試用。如果要傳給朋友裝，要變成 **.apk** 檔。

1. 網頁上方選單 **"Build (打包 APK)"** -> **"Android App (.apk)"**
2. 等進度條跑完 (可能會跑很久，這是在雲端編譯)。
3. 跑完會出現 QR Code 和 下載連結。
   - **QR Code**: 讓手機掃描直接下載安裝。
   - **Download**: 下載檔案存到電腦。

<br>

*註：現在還有 .aab 格式 (Android App Bundle)，是上架 Google Play 專用的。*

# 常見問題與除錯

### 1. 畫面跑版
- 原因：手機螢幕大小不同。
- 解法：多用 **Layout** 和 **Fill Parent / 30%** 這種相對數值，不要用 100px 這種固定數值。

### 2. 程式沒反應 / 當機
- 原因：邏輯錯誤、無窮迴圈、除以零。
- 解法：**Do It** 功能。
  - 連線模式下，對積木按右鍵 -> **"Do It"**。
  - 可以直接看到那塊積木現在的值是多少！(超好用)
