# 課程評分標準

我們使用以下方式進行學期成績計算：

| 項目 | 佔比 | 說明 |
| :--- | :--- | :--- |
| **出席率** | **20%** | 每週點名，請勿遲到早退。 |
| **作業 (Homework)** | **80%** | 共三次主要作業，包含期中作業與期末作業。 |

> **Note**: 這是實作課程，請務必動手練習。
> **Note**: 作業會有繳交期限，學期間的作業若未繳交，可以補交期限為期末，但該作業分數會乘上80%。
