# 課程評分標準

我們使用以下方式進行學期成績計算：

| 項目 | 佔比 | 說明 |
| :--- | :--- | :--- |
| **出席率** | **20%** | 每週點名，請勿遲到早退。 |
| **網頁部分** | **40%** | 共兩次作業。 |
| **Unity應用部分** | **40%** | 依照洪明顯老師的評分標準。 |

> **Note**: 網頁作業會有繳交期限，學期間的作業若未繳交，可以補交期限為期末，但該作業分數會乘上80%。
