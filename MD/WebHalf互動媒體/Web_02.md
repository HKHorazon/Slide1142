# 網站的種類 (以用途來分)


<br>

1. **入口網站 (Portal)**：Yahoo, MSN (資訊匯集站)
2. **電子商務 (E-Commerce)**：PChome, 蝦皮 (買賣東西)
3. **企業形象官網 (Corporate)**：台積電, 弘光科大 (介紹公司)
4. **社群網站 (Social Media)**：FB, IG, Dcard (人際互動)
5. **影音串流 (Streaming)**：YouTube, Netflix (看影片)
6. **個人作品集/部落格 (Portfolio/Blog)**：展示個人專業 (`.md` 轉網站)

# 靜態網頁 vs 動態網頁


<style scoped>
table {
    font-size: 34px;

| 特性 | 靜態網頁 (Static) | 動態網頁 (Dynamic) |
| :--- | :--- | :--- |
| **內容** | 固定不變，所有人看到都一樣 | 根據使用者不同而改變 (個人化) |
| **更新** | 需手動修改 HTML 檔案 | 透過後台系統 (CMS) 修改資料庫 |
| **技術** | HTML + CSS (+ 少許 JS) | HTML/CSS/JS + 資料庫 + 後端語言 |
| **例子** | 個人履歷、形象頁面 | FB 動態牆、購物車、會員系統 |
| **成本** | 低、速度快 | 高、開發複雜 |

# 網站開發流程 

製作一個網站，就像蓋一棟房子，不能想到什麼蓋什麼，需要經過規劃：

1. **目標設定 (Goal Setting)**
   - 網站的目的是什麼？(介紹自己？賣東西？展示作品？)
   - 目標受眾 (Target Audience) 是誰？
   
2. **網站地圖 (Sitemap)**
   - 規劃網站有哪些頁面，以及頁面間的結構 (首頁、關於我、聯絡我們)。

# 傳統架站方式 (Self-Hosting / Cloud)

如果不使用快速架站平台，我們需要自己準備「伺服器」：

1. **自架主機 (On-Premises)**
   - 在家裡或公司準備一台電腦，24小時開機。
   - 需要處理固定 IP、防火牆、不斷電系統等問題。

2. **雲端主機 (Cloud Server)**
   - 向雲端服務商 (AWS, Google Cloud, Azure) 租用虛擬主機。
   - 優點：不用管硬體、隨時可擴充效能、由專人維護機房。
   - 缺點：需要按月付費，且通常需具備 Linux 操作能力。

---

# 常見的伺服器軟體 (Server Software)

要讓電腦變成「網站伺服器」，需要安裝特定的軟體：

### 1. 網頁伺服器 (Web Server)
- **Apache**：老牌、功能完整，市佔率高 (如 XAMPP 內建)。
- **Nginx**：輕量、效能極佳，現代高流量網站首選。
- **IIS**：微軟開發，只能在 Windows Server 上執行。

### 2. 資料庫 (Database)
- **MySQL / MariaDB**：最普及的關聯式資料庫。
- **PostgreSQL**：功能強大，適合複雜的應用程式。

# 快速架站工具 (No-Code/Low-Code)

如果你不會寫程式 (HTML/CSS)，也可以透過「內容管理系統 (CMS)」或「架站平台」來製作網站。

常見的工具有：

1. **Google Sites (Google 協作平台)**
2. **Wix**
3. **WordPress**

---

# 1. Google Sites

最簡單、最快速的選擇。只要有 Google 帳號就能用。

- **優點**：
    - 完全免費，無廣告。
    - 與 Google 服務 (Drive, Docs, Maps) 整合度極高。
    - 操作介面直覺，就像編輯文件一樣。
- **缺點**：
    - 版面模組固定，客製化程度低。
    - 功能較陽春，無法做動態網站(如購物網站)。
- **適合**：個人作品集、活動網頁、班級網頁、內部公告。

---

# 2. Wix

主打「拖拉式 (Drag-and-Drop)」設計，能做出非常精美的動畫與視覺效果。

- **優點**：
    - 設計自由度極高，甚至可以調整元件的像素位置。
    - 內建大量高品質範本 (Templates)。
    - 有 App Market 可以擴充功能。
- **缺點**：
    - 免費版網址醜且有廣告。
    - 網站代碼較雜亂，可能影響載入速度。
    - **平台綁定**：搬家困難，無法輕易匯出到其他伺服器。
- **適合**：設計師作品集、形象官網、小型電商。

---

# 3. WordPress

全世界市佔率最高的架站系統 (超過 40% 網站都用它)。

- **分類**：
    - **WordPress.com** (平台託管)：類似 Wix，有免費版但限制多。
    - **WordPress.org** (自行架設)：需租伺服器，但擁有 100% 控制權。
- **優點**：
    - **強大的生態系**：數萬種外掛 (Plugins) 與 佈景主題 (Themes)。
    - **SEO 友善**：搜尋引擎排名表現通常較好。
    - 可擴充性強 (部落格、電商、論壇都能做)。
- **缺點**：
    - 學習曲線較陡峭 (尤其是自行架設)。
    - 需要定期維護更新，注意資安問題。
- **適合**：部落格、新聞媒體、中大型企業官網、專業電商。
